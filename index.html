Mais detalhes :)

  • O Vue.js tem uma curva de aprendizado extremamente simples se comparado ao Angular 2 e principalmente ao React.
  • Ele tem uma performance surpreendente se equiparado aos mesmos frameworks.
  • Ele é reativo igual ao React e também utiliza Virtual DOM.
  • A comunidade do Vue é espetacular.
  • Existem diversos componentes e plugins já criados.
  • Ou seja, o Vue é que há de melhor na sintaxe bonita e fácil do Angular e na escalabilidade e grandiosidade do React.

Vue.js x React

Sintaxe

React


render ()  {
  let { items } = this.props
  let children
  if  (items.length > 0)  {
    children  = (
      <ul>
        {items.map(item =>
          <li  key={item.id}>{item.name}</li>
        )}
      </ul>
    )
  } else  {
    children  = <p>No  items found.</p>
  }
  return  (
    <div className='list-container'>
      {children}
    </div>
  )
}

Vue.js


<template>
    <div  class="list-container">
        <ul v-if="items.length">
            <li v-for="item in  items">
                {{  item.name }}
            </li>
        </ul>
        <p  v-else>No items found.</p>
    </div>
</template>
